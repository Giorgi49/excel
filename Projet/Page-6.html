Pour associer les âges au code de chaque personne, nous devons d’abord récupérer la position relative d’un code avec EQUIV, puis le prénom associé avec INDEX, ensuite la position relative de ce prénom et enfin l’âge associé en précisant que les plages se trouve dans la feuille “Données”.
